# FrameStudyo

A JavaFX desktop graphic design application inspired by tools like Canva. FrameStudyo lets users create, edit, and export visual designs through an intuitive drag-and-drop interface, with smart assistive features built in.

---

## Features

**Design Tools**
- Drag-and-drop canvas with shape and text editing
- Auto-align for precise element positioning
- Shape recognition to snap freehand drawings to clean geometry
- Template engine with reusable design layouts

**Smart Assistance**
- Color palette suggester based on current design context
- Design scoring to evaluate visual balance and composition

**Analytics and History**
- Session tracking to log design activity
- Design history with shape usage statistics
- Export and save designs locally

---

## Tech Stack

- **Language:** Java
- **UI Framework:** JavaFX
- **IDE:** IntelliJ IDEA
- **Architecture:** Object-Oriented Programming (OOP)

---

## Getting Started

### Prerequisites

- Java 17 or higher
- JavaFX SDK (matching your Java version)
- IntelliJ IDEA (recommended)

### Installation

1. Clone the repository:
   git clone https://github.com/ayesha-bscs25seecs/FrameStudyo.git

2. Open the project in IntelliJ IDEA.

3. Configure the JavaFX SDK under File > Project Structure > Libraries.

4. Run Main.java to launch the application.
